# Roadmap Overview

## 1. Purpose

This document describes the planned development roadmap for the AI Services Website & Automation Platform.

The roadmap is MVP-based and should be updated as project scope, business priorities and technical decisions evolve.

---

## 2. Roadmap Summary

| Order | MVP | Focus |
|---|---|---|
| 1 | MVP 0 | Website, contact form, GCP foundation |
| 2 | MVP 1 | AI chatbot demo |
| 3 | MVP 2 | Lead handling and mail automation |
| 4 | MVP 3 | Multi-client chatbot configuration |
| 5 | MVP 4 | RAG / knowledge base |
| 6 | MVP 5 | Admin panel |
| 7 | MVP 6 | Voice assistant |
| 8 | MVP 7 | Production hardening |

---

## 3. Phase 1 — Public Website and Demo

Includes:

- landing page,
- offer presentation,
- contact form,
- basic backend,
- chatbot demo,
- initial deployment to GCP.

Target outcome:

```text
A potential client can visit the website, understand the offer, test an AI chatbot demo and submit a contact request.
```
